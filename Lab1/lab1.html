Part 1: The Gravitational Potential of the Earth¶

The gravitational potential at a distance $r$ from a mass $M$ is given by:

$$\Phi(r) = -\frac{GM}{r}$$

where G is the gravitational constant.

def gravPot(M : (int, float), x_M : (int, float), y_M : (int, float), x : (int, float, np.ndarray), y : (float, np.ndarray)) -> (float, np.ndarray):
    """A function that returns the gravitational potential at a given point from a body with mass at a given point

    Args:
        M (int, float, 2D array): mass of a body (kg)
        x_M (int, float, 2D array): x coordinate of the mass (m)
        y_M (int, float, 2D array): y coordinate of the mass (m)
        x (int, float, 2D array): x coordinate of the evaluation point (m)
        y (int, float, 2D array): y coordinate of the evaluation point (m)

    Returns: 
        float, 2D array: gravitational potential potential (J/kg)
    """
    if not isinstance(M, (int, float)) or not isinstance(x_M, (int, float))  or not isinstance(y_M, (int, float)) or not (isinstance(x, (int, float)) or isinstance(x, np.ndarray)) or not (isinstance(y, (int, float)) or isinstance(y, np.ndarray)):
        raise TypeError("Input is of wrong type!")
    elif M < 0:
        raise ValueError("Mass cannot be negative!")
    else:
        r = np.sqrt((x_M - x)**2 + (y_M - y)**2)

        mask = (r == 0)

        if np.isscalar(r):
            if r == 0:
                raise ValueError("Evaluation point cannot coincide with the mass!")
        else:
            r = np.where(mask, 1, r)
        
        pot = -G * M / r
        assert isinstance(pot, float) or isinstance(pot, np.ndarray), "Potential isn't a float!"
        assert np.all(pot <= 0), "Potential should never be positive!"
        if not np.isscalar(r):
            pot = np.ma.masked_where(mask, pot)
        return pot

Part 3: The Gravitational Force Field of the Earth-Moon System¶

The gravitational force $\vec{F}$ that a mass $M_1$ exerts on a mass $m_2$ is:

$$\vec{F}_{21} = -G \frac{M_1 m_2}{|\vec{r}_{21}|^2} \hat{r}_{21}$$

where $\vec{r}_{21}$ is the displacement vector from $M_1$ to $m_2$.

def gravForce(M1 : (int, float), m2 : (int, float), x1 : (int, float), y1 : (int, float), x2 : (int, float, np.ndarray), y2 : (int, float, np.ndarray)) -> (float, np.ndarray):
    """A function that returns the x and y components of gravitational force between two masses at given coordinates

    Args:
        M1 (int, float): mass of the first body (kg)
        m2 (int, float): mass of the second body (kg)
        x1 (int, float): x coordinate of the first body (m)
        y1 (int, float): y coordinate of the first body (m)
        x2 (int, float, 2D array): x coordinate of the second body (m)
        y2 (int, float, 2D array): y coordinate of the second body (m)

    Returns: 
        float, 2D array: x component of the gravitational force (N)
        float, 2D array: y component of the gravitational force (N)
    """
    if not isinstance(M1, (int, float)) or not isinstance(m2, (int, float)) or not isinstance(x1, (int, float))  or not isinstance(y1, (int, float)) or not (isinstance(x2, (int, float)) or isinstance(x2, np.ndarray)) or not (isinstance(y2, (int, float)) or isinstance(y2, np.ndarray)):
        raise TypeError("Input is of wrong type!")
    elif M1 < 0 or m2 < 0:
        raise ValueError("Mass cannot be negative!")
    else:
        r2 = (x1 - x2)**2 + (y1 - y2)**2
        
        mask = (r2 == 0)

        if np.isscalar(r2):
            if r2 == 0:
                raise ValueError("Bodies cannot occupy the same position!")
        else:
            r2 = np.where(mask, 1, r2)
        
        Fmag = G * M1 * m2 / r2

        Fx = Fmag * - (x2 - x1) / np.sqrt(r2)
        Fy = Fmag * - (y2 - y1) / np.sqrt(r2)
        
        assert (isinstance(Fx, float) or isinstance(Fx, np.ndarray)) and (isinstance(Fy, float) or isinstance(Fy, np.ndarray)), "Gravitational force isn't a float!"
        
        if not np.isscalar(r2):
            Fx = np.ma.masked_where(mask, Fx)
            Fy = np.ma.masked_where(mask, Fy)
            
        return Fx, Fy

Part 4: Altitude of the Saturn V Rocket¶

Rockets function by conservation of momentum. The ejection of fuel backwards propels the rocket forwards. The change in a rocket's velocity $\Delta v$ as a function of time $t$ (under a set of simplifying assumptions) is given by the Tsiolkovsky rocket equation:

$$\Delta v(t) = v_e \ln \left(\frac{m_0}{m(t)}\right) - gt$$

where $m_0$ is the initial "wet" mass (fuel + rocket parts + payload), $m(t) = m_0 - \dot{m}t$ is the mass at time $t$, $\dot{m}$ is the fuel burn rate (which we will assume to be constant), $v_e$ is the fuel exhaust velocity, and $g$ is the gravitational acceleration.

You can find the altitude $h$ of the rocket at "burnout" (when all the fuel is used up) by integrating the velocity over time:

$$h = \int_{0}^{T} \Delta v(t) \, dt$$

where $T$ is the total burn time of the rocket, which by conservation of momentum arguments is:

$$T = \frac{m_0 - m_f}{\dot{m}}$$

where $m_f$ is the final "dry mass" of the rocket, once all the fuel is burned.

def dvel(t : (int, float, np.ndarray), m0 : (int, float), mf : (int, float), mdot : (int, float), ve : (int, float), grav : (int, float)) -> (float, np.ndarray):
    """A function that returns the change in velocity of a rocket given the time, wet mass, dry mass, burn rate, exhaust speed, and gravitational constant.

    Args:
        t (int, float, 2D array): time (s)
        m0 (int, float): wet mass (kg)
        mf (int, float): dry mass (kg)
        mdot (int, float): burn rate (kg/s)
        ve (int, float): exhaust speed (m/s)
        grav (int, float): gravitational constant (m/s^2)

    Returns: 
        float, 2D array: change in velocity (m/s)
    """
    if not (isinstance(t, (int, float)) or isinstance(t, np.ndarray)) or not isinstance(m0, (int, float)) or not isinstance(mf, (int, float)) or not isinstance(mdot, (int, float)) or not isinstance(ve, (int, float)) or not isinstance(grav, (int, float)):
        raise TypeError("Input is of wrong type!")
    if np.any(t < 0):
        raise ValueError("Time cannot be negative!")
    elif m0 <= 0 or mf <= 0:
        raise ValueError("Mass must be positive!")
    elif mdot <= 0 or ve <= 0:
        raise ValueError("Burn rate, exhaust speed must be positive!")
    elif grav < 0:
        raise ValueError("Gravitational constant cannot be negative!")
    else:
        m = m0 - (mdot * t)
        m_safe = np.where(m <= 0 , np.nan, m)
        
        v = np.where(m <= mf, 0, (ve * np.log(m0 / m_safe)) - grav * t)

        assert (isinstance(v, float) or isinstance(v, np.ndarray) or np.any(np.isnan())), "Change in velocity isn't a float!"
        
        return v
